## Current Focus

Building **[Luffy](https://github.com/Salrahim21/luffy)** — a local-first AI operating system designed to run entirely on-device.

The architecture is a Tauri v2 desktop shell over a FastAPI backend with a multi-agent orchestration engine. It coordinates local LLMs through Ollama, manages persistent context across sessions, and exposes a pluggable skill system for extensibility. The platform includes financial analytics across 11 global stock exchanges, personal knowledge management with spaced repetition, browser automation, and messaging integrations for Telegram and WhatsApp.

Everything runs locally. No cloud dependency. No API keys required for core functionality.

`Python` `TypeScript` `React` `FastAPI` `Tauri v2` `SQLite` `Ollama`
