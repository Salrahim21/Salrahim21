### What I'm Building

**Luffy** — A local-first AI operating system

> Native desktop app (Tauri + React), FastAPI backend, multi-agent orchestration with specialist agents, MarketMind market analytics across 11 global exchanges, personal finance tracking, spaced-repetition learning system, Telegram/WhatsApp bots, browser automation, vision pipelines, and 40+ pluggable skill modules. Everything runs locally with Ollama LLMs.

**Key capabilities:**
- Multi-agent harness with specialist routing (code, research, finance, creative)
- MarketMind: technical analysis, Monte Carlo simulation, sentiment, backtesting
- Wealth OS: portfolio tracking, budgeting, transaction management, PDF reports
- 25-panel desktop Mission Control with real-time WebSocket events
- Telegram + WhatsApp interfaces with async message handling
- SM-2 spaced repetition learning system with video/vision course generation
